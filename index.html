Alexanderize Plugin

Not enough people named Alex on your site? Add a few of ours.

How to Use

  1. Download the Alexanderize Code Kit for all the components you'll need.
  2. Add the images and sounds files to your /images directory.
  3. Make sure you're including jQuery at the end of the body in your HTML file:
    <script src="//ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
  4. Include the individual script files for each Alex you'd like to add to your site:
    <script src="javascripts/jquery.alexFedorovize.js"></script>
<script src="javascripts/jquery.alexStetsonize.js"></script>
<script src="javascripts/jquery.alexSylviate.js"></script>
<script src="javascripts/jquery.alecHarrisonize.js"></script>
  5. Finally, attach each function to an anchor tag or button and reference a unique selector in the call. When a user clicks on the element you've attached an Alex to, they'll come roaring in!
    <script type="text/javascript">
  $(window).load(function() {
    $('a#alexFedorov').alexFedorovize();
    $('a#alexStetson').alexStetsonize();
    $('a#alexSylvia').alexSylviate();
    $('a#alecHarrison').alecHarrisonize();
  });
</script>
